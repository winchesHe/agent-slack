# M2 Slack Render Flow 重设计 实施计划（阶段 4，重写版）

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 M1 的"一条 placeholder 通吃"UX 彻底重做为三载体模型（状态条 + progress message + reply messages），对齐 kagura；同时把 `AgentExecutionEvent` 从 AI SDK 细粒度事件升级为粗粒度 4 类事件，聚合层下沉到 `AiSdkExecutor` 内部。

**Architecture:** 遵循 [`specs/2026-04-19-slack-render-flow-redesign.md`](../specs/2026-04-19-slack-render-flow-redesign.md)。核心分层：
- `AiSdkExecutor`（内部 AggregatorState，对外粗粒度 4 类事件 `activity-state` / `assistant-message` / `usage-info` / `lifecycle`）
- `SlackEventSink`（有状态协调器，持 progressMessageTs / toolHistory / lastStateKey / terminalPhase）
- `SlackRenderer`（无状态门面，每个方法内置 `safeRender`；Slack Assistant feature 首次失败自动降级）
- `ConversationOrchestrator`（消费粗事件，按新 spec §7.1 整批写 jsonl，兜底异常走 `emitSyntheticFailed`）

**Tech Stack:** `ai@4.3.19`、`@ai-sdk/openai-compatible`、`@slack/bolt` + `@slack/web-api`、`markdown-to-slack-blocks@1.5.0`（`markdownToBlocks` + `splitBlocksWithText`）、`vitest`。

**Spec 对照：** 新 spec 全文为权威来源；原架构 spec `specs/2026-04-17-agent-slack-architecture-design.md` §2.2 / §2.3 / §4 / §4.1 / §4.2 / §6.2 / §7.1 被新 spec 覆盖。

**Range:** 对应原 spec §7 阶段 4。**前置**：M1 已完成（Slack 能回复文本、ModelMessage 持久化、memory 注入）。**不包含**：skills / queue / abort（M3）、CLI（M4）；但 abort 走粗事件 `lifecycle { stopped }` 的接口已在本 plan 中落定（M3 只需加触发方）。

```ts
import type { ModelMessage } from 'ai'

// 生命周期阶段
export type LifecyclePhase = 'started' | 'completed' | 'stopped' | 'failed'

// stopped 场景的原因
export type StopReason = 'user' | 'superseded' | 'shutdown'

// 进程态快照：activities 走 Slack loading_messages 池，客户端轮换
export interface ActivityState {
  status: string
  activities: string[]
  composing?: boolean      // 已进入"出文本"阶段
  clear?: boolean          // 清空状态条 + 删 progress message
  newToolCalls?: string[]  // 本次新出现的 tool 名，sink 累加 toolHistory（不参与 key diff）
  reasoningTail?: string   // reasoning 末尾 ≤80 char 摘要
}

// turn-level usage
export interface SessionUsageInfo {
  durationMs: number
  totalCostUSD: number                    // 0 表示 providerMetadata 未能读到 cost
  modelUsage: Array<{
    model: string
    inputTokens: number
    outputTokens: number
    cachedInputTokens: number
    cacheHitRate: number                  // 0–1，total=0 时为 0
  }>
}

// 粗粒度 4 类事件
export type AgentExecutionEvent =
  | { type: 'activity-state'; state: ActivityState }
  | { type: 'assistant-message'; text: string }
  | { type: 'usage-info'; usage: SessionUsageInfo }
  | {
      type: 'lifecycle'
      phase: LifecyclePhase
      reason?: StopReason
      error?: { message: string }
      finalMessages?: ModelMessage[]      // completed 必带；stopped 可带（best-effort）
    }
```

```ts
export const LOADING_POOL = [
  '正在组织思路…',
  '梳理脉络中…',
  '权衡各种角度…',
  '追溯问题根源…',
  '勾勒答案轮廓…',
  '编织片段中…',
  '让答案浮现…',
  '换个角度看看…',
  '仔细品味问题…',
  '寻找合适的措辞…',
  '把碎片连成整体…',
  '专注于关键所在…',
  '在可能性中漫游…',
  '层层构建理解…',
  '感知问题轮廓…',
  '小心地落子…',
  '让思绪沉淀片刻…',
  '从静默中汲取…',
] as const

// Fisher–Yates 洗牌（非原地，返回新数组）
export function getShuffledLoadingMessages(count = 8): string[] {
  const arr = [...LOADING_POOL]
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1))
    ;[arr[i], arr[j]] = [arr[j]!, arr[i]!]
  }
  return arr.slice(0, Math.min(count, arr.length))
}

export const TOOL_PHRASE = {
  input: (name: string) => `准备调用 ${name}…`,
  running: (name: string) => `正在 ${name}…`,
}
```

```ts
// src/im/slack/SlackRenderer.ts
import type { WebClient } from '@slack/web-api'
import type { Logger } from '@/logger/logger.ts'
import type { SessionUsageInfo } from '@/core/events.ts'

export interface SlackRendererDeps {
  logger: Logger
  // 进程级 Slack Assistant feature 可用性开关；缺省认为可用
  assistantFeatureAvailable?: () => boolean
  setAssistantFeatureAvailable?: (v: boolean) => void
}

export interface ProgressUiState {
  status: string
  activities: string[]
  toolHistory: Map<string, number>
  composing?: boolean
  reasoningTail?: string
}

export interface SlackRenderer {
  addAck(client: WebClient, channelId: string, messageTs: string): Promise<void>
  addDone(client: WebClient, channelId: string, messageTs: string): Promise<void>
  addError(client: WebClient, channelId: string, messageTs: string): Promise<void>
  addStopped(client: WebClient, channelId: string, messageTs: string): Promise<void>

  setStatus(
    client: WebClient,
    channelId: string,
    threadTs: string,
    status: string,
    loadingMessages?: string[],
  ): Promise<void>
  clearStatus(client: WebClient, channelId: string, threadTs: string): Promise<void>

  upsertProgressMessage(
    client: WebClient,
    channelId: string,
    threadTs: string,
    state: ProgressUiState,
    prevTs?: string,
  ): Promise<string | undefined>
  finalizeProgressMessageDone(
    client: WebClient,
    channelId: string,
    threadTs: string,
    prevTs: string,
    toolHistory: Map<string, number>,
  ): Promise<void>
  finalizeProgressMessageStopped(
    client: WebClient,
    channelId: string,
    threadTs: string,
    prevTs: string,
  ): Promise<void>
  finalizeProgressMessageError(
    client: WebClient,
    channelId: string,
    threadTs: string,
    prevTs: string,
    errorMessage: string,
  ): Promise<void>
  deleteProgressMessage(
    client: WebClient,
    channelId: string,
    threadTs: string,
    prevTs: string,
  ): Promise<void>

  postThreadReply(
    client: WebClient,
    channelId: string,
    threadTs: string,
    text: string,
    options?: { workspaceLabel?: string },
  ): Promise<void>

  postSessionUsage(
    client: WebClient,
    channelId: string,
    threadTs: string,
    usage: SessionUsageInfo,
  ): Promise<void>
}

export function createSlackRenderer(deps: SlackRendererDeps): SlackRenderer {
  const log = deps.logger.withTag('slack:render')

  // safeRender：所有 Slack API 调用的零负担包装
  async function safeRender<T>(label: string, fn: () => Promise<T>): Promise<T | undefined> {
    try {
      return await fn()
    } catch (err) {
      log.warn(`slack api failed: ${label}`, err)
      return undefined
    }
  }

  async function addReaction(
    client: WebClient,
    channelId: string,
    messageTs: string,
    name: string,
  ): Promise<void> {
    await safeRender(`reactions.add(${name})`, () =>
      client.reactions.add({ channel: channelId, timestamp: messageTs, name }),
    )
  }

  return {
    addAck: (c, ch, ts) => addReaction(c, ch, ts, 'eyes'),
    addDone: (c, ch, ts) => addReaction(c, ch, ts, 'white_check_mark'),
    addError: (c, ch, ts) => addReaction(c, ch, ts, 'x'),
    addStopped: (c, ch, ts) => addReaction(c, ch, ts, 'black_square_for_stop'),

    // 后续 Task 填
    async setStatus() { /* Task 2.2 */ },
    async clearStatus() { /* Task 2.2 */ },
    async upsertProgressMessage() { return undefined /* Task 2.3 */ },
    async finalizeProgressMessageDone() { /* Task 2.3 */ },
    async finalizeProgressMessageStopped() { /* Task 2.3 */ },
    async finalizeProgressMessageError() { /* Task 2.3 */ },
    async deleteProgressMessage() { /* Task 2.3 */ },
    async postThreadReply() { /* Task 2.4 */ },
    async postSessionUsage() { /* Task 2.5 */ },
  }
}
```

```ts
// 内部 helper
function formatUsageLine(usage: SessionUsageInfo): string {
  const parts: string[] = []
  parts.push(`${(usage.durationMs / 1000).toFixed(1)}s`)
  if (usage.totalCostUSD > 0) parts.push(`$${usage.totalCostUSD.toFixed(4)}`)
  for (const m of usage.modelUsage) {
    const nonCachedIn = Math.max(0, m.inputTokens - m.cachedInputTokens)
    const tokens = nonCachedIn + m.outputTokens
    let seg = `${m.model}: ${tokens} non-cached in+out`
    if (m.cacheHitRate > 0) seg += ` (${Math.round(m.cacheHitRate * 100)}% cache)`
    parts.push(seg)
  }
  return parts.join(' · ')
}
```

```ts
/**
 * 按优先级从 providerMetadata 读 USD cost：
 *   1. litellm.cost
 *   2. litellm.response_cost
 *   3. openaiCompat.cost
 * 读不到 → undefined（调用方可回落到 0）
 */
export function extractCostFromMetadata(metadata: unknown): number | undefined {
  if (!metadata || typeof metadata !== 'object') return undefined
  const m = metadata as Record<string, unknown>

  const tryNumber = (v: unknown): number | undefined =>
    typeof v === 'number' && Number.isFinite(v) ? v : undefined

  const paths: Array<() => number | undefined> = [
    () => tryNumber((m.litellm as Record<string, unknown> | undefined)?.cost),
    () => tryNumber((m.litellm as Record<string, unknown> | undefined)?.response_cost),
    () => tryNumber((m.openaiCompat as Record<string, unknown> | undefined)?.cost),
  ]
  for (const p of paths) {
    const v = p()
    if (v !== undefined) return v
  }
  return undefined
}
```

```ts
import { streamText, type LanguageModel, type ModelMessage, type ToolSet } from 'ai'
import type { AgentExecutor, AgentExecutionRequest } from './AgentExecutor.ts'
import type {
  ActivityState,
  AgentExecutionEvent,
  SessionUsageInfo,
} from '@/core/events.ts'
import type { Logger } from '@/logger/logger.ts'
import { extractCostFromMetadata } from './litellm-cost.ts'
import {
  STATUS,
  TOOL_PHRASE,
  getShuffledLoadingMessages,
} from '@/im/slack/thinking-messages.ts'

export interface AiSdkExecutorDeps {
  model: LanguageModel
  tools: ToolSet
  maxSteps: number
  logger: Logger
  modelName?: string
}

interface AggregatorState {
  turnStartedAt: number
  modelUsage: Map<string, {
    inputTokens: number
    outputTokens: number
    cachedInputTokens: number
    costUSD: number
  }>
  lastEmittedActivityKey?: string
  defaultLoadingMessages: string[]
  toolNamesSeenInTurn: Set<string>

  stepTextBuffer: string
  activeTools: Map<string, { toolName: string; status: 'input' | 'running' }>
  composing: boolean

  currentReasoning: string
  lastReasoningEmitAt: number
}

function createAggregator(modelName: string): AggregatorState {
  return {
    turnStartedAt: Date.now(),
    modelUsage: new Map(),
    defaultLoadingMessages: getShuffledLoadingMessages(8),
    toolNamesSeenInTurn: new Set(),
    stepTextBuffer: '',
    activeTools: new Map(),
    composing: false,
    currentReasoning: '',
    lastReasoningEmitAt: 0,
  }
}

// 仅把非 newToolCalls 纳入 key diff；newToolCalls 任何非空都强制重发
function makeActivityKey(state: ActivityState): string {
  const { newToolCalls: _ignore, ...rest } = state
  return JSON.stringify(rest)
}

function* emitActivity(
  agg: AggregatorState,
  state: ActivityState,
): Generator<AgentExecutionEvent> {
  const key = makeActivityKey(state)
  if (
    key === agg.lastEmittedActivityKey &&
    !(state.newToolCalls && state.newToolCalls.length > 0)
  ) {
    return
  }
  agg.lastEmittedActivityKey = key
  yield { type: 'activity-state', state }
}

function buildUsageInfo(agg: AggregatorState): SessionUsageInfo {
  return {
    durationMs: Date.now() - agg.turnStartedAt,
    totalCostUSD: Array.from(agg.modelUsage.values()).reduce((s, u) => s + u.costUSD, 0),
    modelUsage: Array.from(agg.modelUsage.entries()).map(([model, u]) => ({
      model,
      inputTokens: u.inputTokens,
      outputTokens: u.outputTokens,
      cachedInputTokens: u.cachedInputTokens,
      cacheHitRate: u.inputTokens > 0 ? u.cachedInputTokens / u.inputTokens : 0,
    })),
  }
}

function normalizeReasoningTail(s: string): string {
  return s.replace(/\s+/g, ' ').trim().slice(-80)
}

export function createAiSdkExecutor(deps: AiSdkExecutorDeps): AgentExecutor {
  const log = deps.logger.withTag('agent')
  const modelName = deps.modelName ?? 'unknown'

  return {
    async *execute(req: AgentExecutionRequest): AsyncGenerator<AgentExecutionEvent> {
      const agg = createAggregator(modelName)

      // lifecycle started
      yield { type: 'lifecycle', phase: 'started' }
      // 默认 thinking 态
      yield* emitActivity(agg, {
        status: STATUS.thinking,
        activities: agg.defaultLoadingMessages,
      })

      const result = streamText({
        model: deps.model,
        ...(req.systemPrompt ? { system: req.systemPrompt } : {}),
        messages: req.messages,
        tools: deps.tools,
        maxSteps: deps.maxSteps,
        abortSignal: req.abortSignal,
      })

      let encounteredErrorPart = false

      try {
        for await (const part of result.fullStream) {
          const p = part as { type: string; [k: string]: unknown }
          switch (p.type) {
            case 'step-start': {
              agg.stepTextBuffer = ''
              agg.composing = false
              break
            }
            case 'text-delta': {
              const text = String(p.textDelta ?? p.text ?? '')
              agg.stepTextBuffer += text
              if (agg.currentReasoning) agg.currentReasoning = ''
              if (!agg.composing) {
                agg.composing = true
                yield* emitActivity(agg, {
                  status: STATUS.composing,
                  activities: [...agg.defaultLoadingMessages, '正在整理回复…'],
                  composing: true,
                })
              }
              break
            }
            case 'reasoning': {
              const text = String(p.textDelta ?? p.text ?? '')
              agg.currentReasoning += text
              const now = Date.now()
              const cond =
                agg.currentReasoning.length - agg.lastReasoningEmitAt >= 30 ||
                now - agg.lastReasoningEmitAt >= 800
              if (cond) {
                agg.lastReasoningEmitAt = now
                yield* emitActivity(agg, {
                  status: STATUS.reasoning,
                  activities: agg.defaultLoadingMessages,
                  reasoningTail: normalizeReasoningTail(agg.currentReasoning),
                })
              }
              break
            }
            case 'reasoning-signature':
            case 'redacted-reasoning':
            case 'source':
            case 'file':
              break
            case 'tool-call-streaming-start': {
              const callId = String(p.toolCallId ?? '')
              const toolName = String(p.toolName ?? '')
              agg.activeTools.set(callId, { toolName, status: 'input' })
              if (agg.currentReasoning) agg.currentReasoning = ''
              const newly = !agg.toolNamesSeenInTurn.has(toolName)
              if (newly) agg.toolNamesSeenInTurn.add(toolName)
              yield* emitActivity(agg, {
                status: TOOL_PHRASE.running(toolName),
                activities: [
                  TOOL_PHRASE.input(toolName),
                  ...agg.defaultLoadingMessages.slice(0, 4),
                ],
                ...(newly ? { newToolCalls: [toolName] } : {}),
              })
              break
            }
            case 'tool-call-delta':
              break
            case 'tool-call': {
              const callId = String(p.toolCallId ?? '')
              const toolName = String(p.toolName ?? '')
              agg.activeTools.set(callId, { toolName, status: 'running' })
              yield* emitActivity(agg, {
                status: TOOL_PHRASE.running(toolName),
                activities: [TOOL_PHRASE.running(toolName), ...agg.defaultLoadingMessages.slice(0, 4)],
              })
              break
            }
            case 'tool-result': {
              const callId = String(p.toolCallId ?? '')
              agg.activeTools.delete(callId)
              if (agg.activeTools.size === 0) {
                yield* emitActivity(agg, {
                  status: STATUS.thinking,
                  activities: agg.defaultLoadingMessages,
                })
              } else {
                const first = agg.activeTools.values().next().value
                if (first) {
                  yield* emitActivity(agg, {
                    status: TOOL_PHRASE.running(first.toolName),
                    activities: [TOOL_PHRASE.running(first.toolName), ...agg.defaultLoadingMessages.slice(0, 4)],
                  })
                }
              }
              break
            }
            case 'step-finish': {
              const usage = (p.usage ?? {}) as Record<string, number | undefined>
              const meta = p.providerMetadata
              const cost = extractCostFromMetadata(meta) ?? 0
              const inputTokens = usage.promptTokens ?? usage.inputTokens ?? 0
              const outputTokens = usage.completionTokens ?? usage.outputTokens ?? 0
              const cachedInputTokens = usage.cachedInputTokens ?? 0
              const existing = agg.modelUsage.get(modelName) ?? {
                inputTokens: 0,
                outputTokens: 0,
                cachedInputTokens: 0,
                costUSD: 0,
              }
              agg.modelUsage.set(modelName, {
                inputTokens: existing.inputTokens + inputTokens,
                outputTokens: existing.outputTokens + outputTokens,
                cachedInputTokens: existing.cachedInputTokens + cachedInputTokens,
                costUSD: existing.costUSD + cost,
              })
              const trimmed = agg.stepTextBuffer.trim()
              if (trimmed) yield { type: 'assistant-message', text: trimmed }
              agg.stepTextBuffer = ''
              agg.composing = false
              break
            }
            case 'error': {
              encounteredErrorPart = true
              yield {
                type: 'lifecycle',
                phase: 'failed',
                error: { message: String((p as { error?: unknown }).error ?? 'unknown') },
              }
              // 提前 break：不再消费后续 part，也不 emit finish / usage
              break
            }
            case 'finish':
              break
            default:
              break
          }
          if (encounteredErrorPart) break
        }

        if (!encounteredErrorPart) {
          const response = await result.response
          const finalMessages = response.messages as ModelMessage[]
          yield { type: 'usage-info', usage: buildUsageInfo(agg) }
          yield {
            type: 'lifecycle',
            phase: 'completed',
            finalMessages,
          }
        }
      } catch (err) {
        const isAbort =
          err instanceof Error &&
          (err.name === 'AbortError' || err.message.toLowerCase().includes('abort'))
        if (isAbort) {
          yield { type: 'activity-state', state: { status: '', activities: [], clear: true } }
          // best-effort 读取已完成 step 的 messages
          const response = await result.response.catch(() => undefined)
          const best = (response as { messages?: ModelMessage[] } | undefined)?.messages
          yield {
            type: 'lifecycle',
            phase: 'stopped',
            reason: 'user',
            ...(best && best.length > 0 ? { finalMessages: best } : {}),
          }
        } else {
          log.error('executor stream error', err)
          yield {
            type: 'lifecycle',
            phase: 'failed',
            error: { message: err instanceof Error ? err.message : String(err) },
          }
        }
      }
    },
  }
}
```

```ts
// src/im/slack/SlackEventSink.ts
import type { WebClient } from '@slack/web-api'
import type { Logger } from '@/logger/logger.ts'
import type {
  ActivityState,
  AgentExecutionEvent,
  LifecyclePhase,
  SessionUsageInfo,
  StopReason,
} from '@/core/events.ts'
import type { EventSink } from '@/im/types.ts'
import type { SlackRenderer } from './SlackRenderer.ts'
import { STATUS, getShuffledLoadingMessages } from './thinking-messages.ts'

export interface SlackEventSinkDeps {
  web: WebClient
  channelId: string
  threadTs: string
  sourceMessageTs: string
  workspaceLabel?: string
  renderer: SlackRenderer
  logger: Logger
}

export interface SlackEventSink extends EventSink {
  onEvent(event: AgentExecutionEvent): Promise<void>
  finalize(): Promise<void>
  readonly terminalPhase: LifecyclePhase | undefined
}

interface SinkLocalState {
  progressMessageTs?: string
  progressActive: boolean
  toolHistory: Map<string, number>
  lastStateKey?: string
  hasSentToolbarInTurn: boolean
  terminalPhase?: LifecyclePhase
  terminalStopReason?: StopReason
  terminalErrorMessage?: string
  pendingUsage?: SessionUsageInfo
}

function makeStateKey(s: ActivityState): string {
  const { newToolCalls: _i, ...rest } = s
  return JSON.stringify(rest)
}

function isMeaningful(state: ActivityState): boolean {
  if (state.clear) return false
  if (state.composing) return true
  if (state.reasoningTail) return true
  if (state.status && state.status !== STATUS.thinking) return true
  if (state.newToolCalls && state.newToolCalls.length > 0) return true
  return false
}

export function createSlackEventSink(deps: SlackEventSinkDeps): SlackEventSink {
  const log = deps.logger.withTag('slack:sink')
  const local: SinkLocalState = {
    progressActive: false,
    toolHistory: new Map(),
    hasSentToolbarInTurn: false,
  }

  async function handleActivity(state: ActivityState): Promise<void> {
    // 累加 toolHistory（不影响 key diff）
    if (state.newToolCalls && state.newToolCalls.length > 0) {
      for (const name of state.newToolCalls) {
        local.toolHistory.set(name, (local.toolHistory.get(name) ?? 0) + 1)
      }
    }

    // clear 分支：删 progress + clearStatus
    if (state.clear) {
      if (local.progressMessageTs) {
        await deps.renderer.deleteProgressMessage(
          deps.web,
          deps.channelId,
          deps.threadTs,
          local.progressMessageTs,
        )
        local.progressMessageTs = undefined
        local.progressActive = false
      }
      await deps.renderer.clearStatus(deps.web, deps.channelId, deps.threadTs)
      local.lastStateKey = undefined
      return
    }

    // key diff 去重（仅在 newToolCalls 为空时）
    const hasNewTools = state.newToolCalls && state.newToolCalls.length > 0
    const key = makeStateKey(state)
    if (!hasNewTools && key === local.lastStateKey) return
    local.lastStateKey = key

    if (local.progressMessageTs) {
      const ts = await deps.renderer.upsertProgressMessage(
        deps.web,
        deps.channelId,
        deps.threadTs,
        {
          status: state.status,
          activities: state.activities,
          toolHistory: local.toolHistory,
          ...(state.composing ? { composing: true } : {}),
          ...(state.reasoningTail ? { reasoningTail: state.reasoningTail } : {}),
        },
        local.progressMessageTs,
      )
      if (ts) local.progressMessageTs = ts
      return
    }

    if (isMeaningful(state)) {
      // 让状态条让位
      await deps.renderer.clearStatus(deps.web, deps.channelId, deps.threadTs)
      const ts = await deps.renderer.upsertProgressMessage(
        deps.web,
        deps.channelId,
        deps.threadTs,
        {
          status: state.status,
          activities: state.activities,
          toolHistory: local.toolHistory,
          ...(state.composing ? { composing: true } : {}),
          ...(state.reasoningTail ? { reasoningTail: state.reasoningTail } : {}),
        },
      )
      if (ts) {
        local.progressMessageTs = ts
        local.progressActive = true
      }
      return
    }

    // 仅刷状态条
    const loadingMessages =
      state.reasoningTail
        ? [...state.activities, `🤔 ${state.reasoningTail}`]
        : state.activities
    await deps.renderer.setStatus(
      deps.web,
      deps.channelId,
      deps.threadTs,
      state.status,
      loadingMessages,
    )
  }

  async function handleAssistantMessage(text: string): Promise<void> {
    const opts = !local.hasSentToolbarInTurn && deps.workspaceLabel
      ? { workspaceLabel: deps.workspaceLabel }
      : undefined
    await deps.renderer.postThreadReply(
      deps.web,
      deps.channelId,
      deps.threadTs,
      text,
      opts,
    )
    local.hasSentToolbarInTurn = true

    if (local.progressMessageTs) {
      await deps.renderer.deleteProgressMessage(
        deps.web,
        deps.channelId,
        deps.threadTs,
        local.progressMessageTs,
      )
      local.progressMessageTs = undefined
      local.progressActive = false
    }
    local.lastStateKey = undefined

    // 下一 step 继续有状态
    await deps.renderer.setStatus(
      deps.web,
      deps.channelId,
      deps.threadTs,
      STATUS.thinking,
      getShuffledLoadingMessages(8),
    )
  }

  async function handleLifecycle(event: Extract<AgentExecutionEvent, { type: 'lifecycle' }>): Promise<void> {
    if (event.phase === 'started') {
      await deps.renderer.addAck(deps.web, deps.channelId, deps.sourceMessageTs)
      await deps.renderer.setStatus(
        deps.web,
        deps.channelId,
        deps.threadTs,
        STATUS.thinking,
        getShuffledLoadingMessages(8),
      )
      return
    }
    // 终态：first-write-wins
    if (local.terminalPhase) {
      log.warn('terminalPhase 已设置，忽略重复终态事件', {
        existing: local.terminalPhase,
        incoming: event.phase,
      })
      return
    }
    local.terminalPhase = event.phase
    if (event.phase === 'stopped') local.terminalStopReason = event.reason
    if (event.phase === 'failed') local.terminalErrorMessage = event.error?.message ?? 'unknown'
  }

  return {
    async onEvent(event) {
      try {
        switch (event.type) {
          case 'activity-state':
            await handleActivity(event.state)
            break
          case 'assistant-message':
            await handleAssistantMessage(event.text)
            break
          case 'usage-info':
            local.pendingUsage = event.usage
            break
          case 'lifecycle':
            await handleLifecycle(event)
            break
        }
      } catch (err) {
        log.error('sink onEvent 内部异常（不冒泡）', err)
      }
    },
    async finalize() {
      try {
        await deps.renderer.clearStatus(deps.web, deps.channelId, deps.threadTs)
        if (local.progressMessageTs) {
          const prev = local.progressMessageTs
          if (local.terminalPhase === 'completed') {
            await deps.renderer.finalizeProgressMessageDone(
              deps.web,
              deps.channelId,
              deps.threadTs,
              prev,
              local.toolHistory,
            )
          } else if (local.terminalPhase === 'stopped') {
            if (local.terminalStopReason === 'superseded') {
              await deps.renderer.deleteProgressMessage(
                deps.web,
                deps.channelId,
                deps.threadTs,
                prev,
              )
            } else {
              await deps.renderer.finalizeProgressMessageStopped(
                deps.web,
                deps.channelId,
                deps.threadTs,
                prev,
              )
            }
          } else if (local.terminalPhase === 'failed') {
            await deps.renderer.finalizeProgressMessageError(
              deps.web,
              deps.channelId,
              deps.threadTs,
              prev,
              local.terminalErrorMessage ?? 'unknown',
            )
          }
          local.progressMessageTs = undefined
          local.progressActive = false
        }

        if (local.terminalPhase === 'completed' && local.pendingUsage) {
          await deps.renderer.postSessionUsage(
            deps.web,
            deps.channelId,
            deps.threadTs,
            local.pendingUsage,
          )
        }

        if (local.terminalPhase === 'completed') {
          await deps.renderer.addDone(deps.web, deps.channelId, deps.sourceMessageTs)
        } else if (local.terminalPhase === 'stopped') {
          await deps.renderer.addStopped(deps.web, deps.channelId, deps.sourceMessageTs)
        } else if (local.terminalPhase === 'failed') {
          await deps.renderer.addError(deps.web, deps.channelId, deps.sourceMessageTs)
        }
      } catch (err) {
        log.error('sink finalize 内部异常（不冒泡）', err)
      }
    },
    get terminalPhase() {
      return local.terminalPhase
    },
  }
}
```

```ts
// src/orchestrator/ConversationOrchestrator.ts（diff 思路，保留 session/memory 逻辑）
import type { AgentExecutor } from '@/agent/AgentExecutor.ts'
import type { SessionStore } from '@/store/SessionStore.ts'
import type { MemoryStore } from '@/store/MemoryStore.ts'
import type { InboundMessage, EventSink } from '@/im/types.ts'
import type { Logger } from '@/logger/logger.ts'
import type { ToolSet, ModelMessage } from 'ai'
import { emitSyntheticFailed } from './emitSyntheticFailed.ts'

// ... deps / factory 不变，以下仅 handle() 主体

async handle(input, sink) {
  const currentUser = { userName: input.userName, userId: input.userId }

  const session = await deps.sessionStore.getOrCreate({ /* ... */ })
  await deps.sessionStore.setStatus(session.id, 'running')

  const history = await deps.sessionStore.loadMessages(session.id)
  const userMsg: ModelMessage = { role: 'user', content: input.text }
  await deps.sessionStore.appendMessage(session.id, userMsg)

  const hasMemory = await deps.memoryStore.exists(currentUser.userName, currentUser.userId)
  const memoryPath = deps.memoryStore.pathFor(currentUser.userName, currentUser.userId)
  const memoryHint = /* ... 同现有 ... */
  const systemPromptWithMemory = `${deps.systemPrompt}${memoryHint}`

  const tools = deps.toolsBuilder(currentUser)
  const executor = deps.executorFactory(tools)

  const ctrl = new AbortController()
  try {
    try {
      for await (const event of executor.execute({
        systemPrompt: systemPromptWithMemory,
        messages: [...history, userMsg],
        abortSignal: ctrl.signal,
      })) {
        await sink.onEvent(event)

        if (event.type === 'usage-info') {
          // 把 usage 落到 meta.json：tokens 按 modelUsage 逐个累加，
          // costUSD 只按整个 turn 的 totalCostUSD 累加一次（防 N 倍放大）。
          for (const u of event.usage.modelUsage) {
            await deps.sessionStore.accumulateUsage(session.id, {
              inputTokens: u.inputTokens,
              outputTokens: u.outputTokens,
              cachedInputTokens: u.cachedInputTokens,
            })
          }
          if (event.usage.totalCostUSD > 0) {
            await deps.sessionStore.accumulateCost(session.id, event.usage.totalCostUSD)
          }
        }

        if (event.type === 'lifecycle') {
          if (event.phase === 'completed') {
            for (const m of event.finalMessages ?? []) {
              await deps.sessionStore.appendMessage(session.id, m)
            }
            await deps.sessionStore.setStatus(session.id, 'idle')
          } else if (event.phase === 'stopped') {
            if (event.finalMessages && event.finalMessages.length > 0) {
              for (const m of event.finalMessages) {
                await deps.sessionStore.appendMessage(session.id, m)
              }
            }
            await deps.sessionStore.appendMessage(session.id, {
              role: 'assistant',
              content: '[stopped]',
            })
            await deps.sessionStore.setStatus(session.id, 'stopped')
          } else if (event.phase === 'failed') {
            await deps.sessionStore.appendMessage(session.id, {
              role: 'assistant',
              content: `[error: ${event.error?.message ?? 'unknown'}]`,
            })
            await deps.sessionStore.setStatus(session.id, 'error')
          }
        }
      }
    } catch (err) {
      log.error('orchestrator handle 内部异常', err)
      await emitSyntheticFailed(sink, err instanceof Error ? err.message : String(err))
      await deps.sessionStore.appendMessage(session.id, {
        role: 'assistant',
        content: `[error: ${err instanceof Error ? err.message : String(err)}]`,
      })
      await deps.sessionStore.setStatus(session.id, 'error')
    }
  } finally {
    await sink.finalize()
  }
}
```

> 需要在 `SessionStore` 中：
> - 添加 `status = 'stopped'` 的支持（若尚未有）
> - `accumulateUsage(sessionId, { inputTokens, outputTokens, cachedInputTokens })` 签名移除 `costUSD`（合并到 session total cost 的单独方法）
> - 新增 `accumulateCost(sessionId, costUSD: number)`，当次 turn 的 `totalCostUSD` 只累加一次
>
> **为什么**：`SessionUsageInfo.modelUsage[number]` 本身没有 per-model `costUSD` 字段（见 spec §2），若在循环内每次用 `totalCostUSD` 会被 N 倍累加（N = modelUsage.length）。

```ts
    expect(msgs[0]).toMatchObject({ role: 'user' })
```

## 风险与注意事项

- **AI SDK v4 API 形态**：`result.response` 是 Promise<{ messages: ModelMessage[] }>；`fullStream` 的 part 名已在新 spec §6.2 核实；但 `MockLanguageModelV1` 的 `doStream` 返回 `LanguageModelV1StreamPart` 是**底层 provider stream**，而非 `streamText().fullStream`（经 AI SDK 包装后的）。测试细节可能需要调整 mock 路径——若遇到：改为 mock 更高层（直接传完整 `result` 对象），或参考 `ai/test` 最新 doc。
- **`@slack/web-api` 的 assistant.threads.setStatus 参数名**：本 plan 假设 `channel_id` / `thread_ts`；若实际 SDK 版本签名不同，以 runtime 为准。
- **AbortError**：AI SDK v4 在 abort 时抛出的错误可能不是原生 `AbortError`，而是带 `name: 'AbortError'` 的普通 Error 或自定义 AISDKError；本 plan 通过 name + message 双判断兼容。
- **cost 多路径**：真实 LiteLLM 响应 shape 需在实装 Task 3.1 时打印一次 `providerMetadata` 定型；若三条路径都读不到，修订 `extractCostFromMetadata` 再跑 Task 3.1 的 test。
- **`splitBlocksWithText` 返回结构**：已核实 export，但运行时结构（`{ blocks, text }[]`）需以真实调用一次为准；若字段名不同，调整 `postThreadReply` 实装。
